Educational Mode

1. Start from the known plan and elevation. Identify the edge or line that must appear true length.
2. Place the auxiliary reference line parallel or perpendicular to the controlling view condition.
3. Project perpendicular rays from the parent view to the auxiliary plane.
4. Transfer distances from the adjacent orthographic view using folding construction.
5. Join corresponding points. The resulting view is tested for true shape, edge view, or point view.

Teaching Notes

Use the sliders to put the lamina into special positions. A plane shows true shape when the viewing direction is normal to the plane. It shows edge view when the viewing direction lies in the plane. A line shows point view when the viewing direction is parallel to that line.
